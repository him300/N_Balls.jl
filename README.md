# N_Balls
N_Balls is not a registered julia package so use "https://github.com/him300/N_Balls.jl.git" to install it.

n_ball_volume(r::Float64,n::Int64) to get the volume of a n-dimensional ball with radius r. 
Vector version: n_ball_volume(r::Vector{Float64},n::Int64) to get a vector of volumes.

n_ball_surface(r::Float64,n::Int64) to get the surface of a n-dimensional ball with radius r. 
Vector version : n_ball_surface(r::Vector{Float64},n::Int64) to get a vector of surfaces.

n_ball_vol_surf(r::Float64,n::Int64) to get the volume and surface of n-dimensional ball with radius r. 
Vector version : n_ball_vol_surf(r::Vector{Float64},n::Int64) to get a DataFrame of both surfaces and volumes.
